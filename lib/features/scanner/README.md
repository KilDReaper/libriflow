# QR/Barcode Scanner Feature

## Overview
The QR Scanner feature allows users to quickly scan book QR codes, barcodes, and ISBN numbers for borrowing, returning, or adding books to the library.

## Usage
Users can access the scanner through:
1. Quick Scan banner on the Dashboard
2. QR icon in the Dashboard app bar
3. Floating "Scan" button on Dashboard, Search, and Library tabs

## Supported Formats
- QR Codes
- Barcodes (EAN, UPC, Code 128, etc.)
- ISBN numbers

## Next Steps
To make the scanner fully functional, connect it to:
- Book borrowing/return API endpoints
- Book lookup service (ISBN to book details)
- User borrowing history
- Notification system for due dates
